<p>Last Week in Security is a summary of the interesting cybersecurity news, techniques, tools and exploits from the previous week. This post covers 2020-11-16 to 2020-11-23.</p>
<section id="news">
<h2>News</h2>
<ul>
<li><a href="https://specterops.io/so-con2020" target="_blank">SO-CON 2020</a> was last Friday and had a ton of good talks. Recordings are available on the conference site for 30 days, and YouTube after that.</li>
<li><a href="https://www.wsj.com/articles/airbnb-executive-resigned-last-year-over-chinese-request-for-more-data-sharing-11605896753" target="_blank">Airbnb Executive Resigned Last Year Over Chinese Request for More Data Sharing</a>. AirBnB is sharing booking data from the moment a reservation is made with China. This gives a lot of lead time to "set up" an apartment or house for surveillance. Any large tech company will soon have to face this decision of what to do about Chinese requests. We have seen how anyone who is unwilling to cooperate is blocked, and then an internal Chinese only copy is propped up.</li>
<li><a href="https://blog.mozilla.org/security/2020/11/17/firefox-83-introduces-https-only-mode/" target="_blank">Firefox 83 introduces HTTPS-Only Mode</a>. Long time users of the <a href="https://addons.mozilla.org/en-US/firefox/addon/https-everywhere/" target="_blank">HTTPS Everywhere</a> extension from the EFF will be happy to have this feature available in the browser itself. This feature attempts to load all pages and resources via HTTPS first, instead of the usual HTTP request, 304 redirect, and HTTPS request flow normally seen. There may also be a speed up as one less request/response has to be processed. Chrome has no plans to implement a similar feature.</li>
<li><a href="https://github.blog/2020-11-16-standing-up-for-developers-youtube-dl-is-back/" target="_blank">Standing up for developers: youtube-dl is back</a>. I didn't expect this. Github/Microsoft has reinstated youtube-dl forks that remove the tests using copyright protected material. They have also changed their DCMA takedown process and created a developer defense fund. This is probably the best response I could have hoped for, and while I'm still leery of Microsoft's ultimate intentions with GitHub, for now at least it is a positive relationship.</li>
<li><a href="https://www.kali.org/news/kali-linux-2020-4-release/" target="_blank">Kali Linux 2020.4 Release</a>. ZSH is now the default shell, a few new tools and version bumps, but the coolest feature is the <a href="https://porchetta.industries/2020/11/17/And-Now-For-Something-Completely-Diffrent/" target="_blank">private partnership with @byt3bl33d3r</a> of the amazing <a href="https://github.com/byt3bl33d3r/CrackMapExec" target="_blank">CrackMapExec</a> exclusive to Kali and GitHub sponsors for 30 days after each release.</li>
<li><a href="https://twitter.com/iangcarroll/status/1329265104882343937" target="_blank">ARM-based macOS can run iOS apps + network traffic/cert store is tied to macOS = perfect for iOS app hacking</a>. This isn't really a new capability, since you could proxy web traffic through Burp on a macOS already. Having it all one one machine makes things slightly easier I suppose.</li>
<li><a href="https://zerossl.com/features/acme/" target="_blank">ZeroSSL offers free TLS certificates</a>. Just like Let'sEncrypt, ZeroSSL now offers free 90 days certificates via the ACME protocol, including wildcard certificates.</li>
</ul>
</section>
<section id="techniques">
<h2>Techniques</h2>
<ul>
<li><a href="https://www.armorblox.com/blog/ok-google-build-me-a-phishing-campaign/" target="_blank">OK Google, Build Me a Phishing Campaign</a>. Use of legitimate services for phishing can be effective, as the domains are likely allowed by most enterprise filters.</li>
<li><a href="https://engineering.salesforce.com/easily-identify-malicious-servers-on-the-internet-with-jarm-e095edac525a" target="_blank">Easily Identify Malicious Servers on the Internet with JARM</a>. Salesforce takes its TLS fingerprinting to the next level by actively "scanning" and categorizing The Alexa top 1 million websites as well as known C2 servers to collect JARM fingerprints. Using these, blue teams can enrich data, or even build blocklists. Advanced red teams should be testing their own infrastructure with <a href="https://github.com/salesforce/jarm" target="_blank">JARM</a> to make sure it blends in. A Go implementation is available here: <a href="https://github.com/RumbleDiscovery/jarm-go" target="_blank">jarm-go</a>.</li>
<li><a href="https://www.archcloudlabs.com/projects/dumping-memory-with-av/" target="_blank">Dumping Memory with AV - Avast Home Security</a>. The use of AV for red team goals is a special kind of sneaky. The best part of these tricks is the binaries are signed by the AV vendors and <a href="https://www.virustotal.com/gui/file/2f223daf4a6d4ff52d53c8f532a780760411f9168ba95cb9937b8ee11e4d9b75/detection" target="_blank">categorized as benign by other AVs</a>!</li>
<li><a href="https://www.fireeye.com/blog/threat-research/2020/11/purgalicious-vba-macro-obfuscation-with-vba-purging.html" target="_blank">Purgalicious VBA: Macro Obfuscation With VBA Purging</a> The opposite of <a href="https://github.com/outflanknl/EvilClippy" target="_blank">EvilClippy</a> which stomps VBA code leaving only the P-cache, this technique stomps the SRP stream (P-cache) removing lots of strings many AV engines and YARA rules flag as malicious. The tool, <a href="https://github.com/fireeye/OfficePurge" target="_blank">OfficePurge</a>, is available now.</li>
<li><a href="https://blog.nviso.eu/2020/11/20/dynamic-invocation-in-net-to-bypass-hooks/" target="_blank">Dynamic Invocation in .NET to bypass hooks</a> walks through the concepts of the standard P/invoke and the tricky D/invoke for C# calls to the Windows API. It covers four techniques for bypassing userland hooks, which could prove useful for EDR bypasses.</li>
<li><a href="https://offensivedefence.co.uk/posts/module-stomping/" target="_blank">Module Stomping in C#</a> is an implementation of the Module Stomping (aka Module Overloading aka DLL Hollowing) shellcode injection technique in C#.</li>
<li><a href="https://secbytes.net/Implant-Roulette-Part-1:-Nimplant" target="_blank">Implant Roulette Part 1: Nimplant</a>. Nim is an interesting language for payload development. It has the ability to output a variety of other languages (i.e. C, C++) and can cross-compile with small binaries. It also has easy to use obfuscation packages. An example implant (<a href="https://github.com/MythicAgents/Nimplant" target="_blank">Nimplant</a>) for Mythic is available. Expect to see more Nim for offensive security projects in the future. Get more Nim goodness at: <a href="https://github.com/byt3bl33d3r/OffensiveNim" target="_blank">OffensiveNim</a>.</li>
<li><a href="https://adepts.of0x.cc/vba-tools/" target="_blank">Shedding light on creating VBA macros</a>. VBA is a strange and difficult language. This post has some tricks to aid in doing low level API work with VBA.</li>
<li><a href="https://ooloo.io/project/github-flow" target="_blank">"I have no idea how to use Git in a professional dev team."</a> This course is great for anyone who has struggled with the complexities of Git. While the example project is a simple website, the concepts are generally applicable to all types of software development. The project comes with a bot that simulates a team, and it's free!</li>
<li><a href="https://medium.com/red-teaming-with-a-blue-team-mentaility/making-objective-c-calls-from-python-standard-libraries-550ed3a30a30" target="_blank">Making Objective C Calls From Python Standard Libraries (Red Team Edition)</a>. With Python 2 still included in the base install of macOS 11.0, the ability to call macOS APIs from python is a useful technique for a red teamer. This research is what lead to the development of <a href="https://github.com/cedowens/MacC2" target="_blank">MacC2</a>.</li>
<li><a href="https://www.offensive-security.com/offsec/microsoft-teams-macos-local-privesc/" target="_blank">Microsoft Teams for macOS Local Privilege Escalation</a>. XPC strikes again! Checking the PID of the sending process is not enough to verify it is a trusted client.</li>
<li><a href="https://insert-script.blogspot.com/2020/11/imagemagick-shell-injection-via-pdf.html" target="_blank">ImageMagick - Shell injection via PDF password</a>. File parsing is hard, and when it goes wrong things can go bad. The ImageMagick is <a href="https://imagetragick.com/" target="_blank">no stranger to attacks</a>, and this most recent issue highlights the dangers of parsing untrusted input.</li>
<li><a href="https://www.mdsec.co.uk/2020/11/a-fresh-outlook-on-mail-based-persistence/" target="_blank">A Fresh Outlook on Mail Based Persistence</a>. Like the other Office applications, Outlook can run VBA scripts and has triggers that will run functions when email is received. All it takes is some simple string comparisons, and you can email your way to beacons from the target to regain access. Nice!</li>
</ul>
</section>
<section id="tools-and-exploits">
<h2>Tools and Exploits</h2>
<ul>
<li><a href="https://wordlists.assetnote.io/" target="_blank">Assetnote Wordlists</a>. When performing security testing against an asset, it is vital to have high quality wordlists for content and subdomain discovery. This website provides you with wordlists that are up to date and effective against the most popular technologies on the internet, generated fresh each month!</li>
<li><a href="https://github.com/prisma-cloud/IAMFinder" target="_blank">IAMFinder</a> enumerates and finds users and IAM roles in a target AWS account. With only the AWS account number of the targeted account, IAMFinder is able to identify users and roles in that environment. Upon successfully identifying an IAM role, IAMFinder can also check if this role can be assumed anonymously. The tool was developed during a red team exercise and it implemented the technique described in <a href="https://unit42.paloaltonetworks.com/aws-resource-based-policy-apis/" target="_blank">this blog</a>.</li>
<li><a href="https://posts.specterops.io/ghostwriter-v2-0-release-638cef16deb7" target="_blank">Ghostwriter v2.0 Release</a>. Ghostwritter is becoming a serious red team management tool. If you haven't looked into it before, it has some great new features that may help your team's workflow. There are adaptors for CobaltStrike and other tools like <a href="https://github.com/hausec/CobaltStrikeToGhostWriter" target="_blank">CobaltStrikeToGhostWriter</a>.</li>
<li><a href="https://github.com/BloodHoundAD/BloodHound/releases/tag/4.0.0" target="_blank">BloodHound 4.0 - Azurehound</a>. This is a major feature release for BloodHound, including support for Azure attack primitives in the attack graph with new nodes and edges. There is a nice <a href="https://hausec.com/2020/11/23/azurehound-cypher-cheatsheet/" target="_blank">cheatsheet</a> for the new Azure functionality.</li>
<li><a href="https://github.com/slyd0g/SwiftSpy" target="_blank">SwiftSpy</a> is a macOS keylogger, clipboard monitor, and screenshotter written in Swift. Be aware it will cause TCC (Transparency, Control, and Consent) popups!</li>
<li><a href="https://github.com/NVISO-BE/DInvisibleRegistry" target="_blank">DInvisibleRegistry</a> is an implementation of the null byte Run key persistence technique implemented in C# with direct syscalls via D/invoke.</li>
<li><a href="https://githacks.org/_xeroxz/vdm" target="_blank">VDM</a> is a library to manipulate drivers exposing a physical memory read/write primitive to allow the user to call any function in the kernel. Currently the project is using gdrv.sys but can be adapted to use any driver that allows for physical memory read and write by writing 4 wrapper functions.</li>
<li><a href="https://github.com/rmdavy/HeapsOfFun" target="_blank">HeapsOfFun</a>, an AMSI VBA bypass via the heap, gets an update for x64. Details <a href="https://secureyourit.co.uk/wp/2020/04/17/amsi-in-the-heap/" target="_blank">here</a>.</li>
<li><a href="https://github.com/theflakes/reg_hunter" target="_blank">reg_hunter</a> is a blueteam operational triage registry hunting/forensic tool written in Rust.</li>
<li><a href="https://github.com/evilpenguin/MachoDecrypt" target="_blank">MachoDecrypt</a> will decrypt mach-o binaries on iOS. Requires a jailbroken iPhone.</li>
<li><a href="https://github.com/Cgboal/exclude-cdn" target="_blank">exclude-cdn</a> a tool to filter out CDN hosts from a list consisting of IP's, URL's, and Domains passed via stdin. Useful for bug bounties or external penetration tests.</li>
<li><a href="https://github.com/klinix5/Kaspersky_Safe_Money_LPE" target="_blank">Kaspersky_Safe_Money_LPE</a> is an 0day in the Kaspersky "Safe Money" protected browser. Pure AV schadenfreude. Demo <a href="https://youtu.be/DbuiFs6_oTo" target="_blank">here</a>.</li>
</ul>
