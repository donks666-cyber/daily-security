<p>Last Week in Security is a summary of the interesting cybersecurity news, techniques, tools and exploits from the previous week. This post covers 2020-03-23 to 2020-03-30.
MITRE ATT&amp;CK techniques are in brackets where appropriate.</p>
<p>Need a project while working from home? <a href="https://twitter.com/Random_Robbie" target="_blank">@Random_Robbie</a> published a <a href="https://gist.github.com/random-robbie/dd6b4a5488609a8fe5326620858cdecd" target="_blank">list of WordPress plugins that call shell_exec</a>. Have fun!</p>
<section id="news">
<h2>News</h2>
<ul>
<li><a href="https://protonvpn.com/blog/apple-ios-vulnerability-disclosure/" target="_blank">VPN bypass vulnerability in Apple iOS</a> - A ProtonVPN community member did his own investigation and realized that iOS does not kill existing connections when a VPN is connected, which could allow long-lived connections such as Apples Push Notification Service to continue outside the VPN. To mitigate, toggle Airplane Mode once the VPN is connect to force all connections to drop and reconnect. This issues is not present in corporate Always-on VPN profiles delivered via device management.</li>
<li><a href="https://www.zdnet.com/article/rare-badusb-attack-detected-in-the-wild-against-us-hospitality-provider/" target="_blank">Rare BadUSB attack detected in the wild against US hospitality provider</a> show that APTs are trying all avenues of attack. In this case a USB and gift card were mailed to a business with a letter enticing the user to open a product list on the USB. However, the USB was a keystroke injector which used PowerShell to download a JScript-based bot. If USB drops/BadUSB attacks aren't part of your adversary emulation offerings, they should be.</li>
<li><a href="https://www.vice.com/en_us/article/wxeykb/phineas-fisher-says-they-paid-dollar10000-bounty-to-person-who-hacked-chilean-military" target="_blank">Phineas Fisher Says They Paid $10,000 Bounty to Person Who Hacked Chilean Military</a> This is the first payout of the "Hacktivist Bug Hunting Program," and interestingly Phineas reached out to the hackers to offer a bounty. On the lack of hacktivist activity recently Phineas said, "there's a time for everything, a time to hack will come."</li>
<li><a href="https://www.whitehouse.gov/wp-content/uploads/2020/03/National-Strategy-5G-Final.pdf" target="_blank">Whitehouse releases 5G security strategy</a> and keeps it short and high level. A "much more detailed" implementation plan is said to be on the way, according to <a href="https://www.nextgov.com/emerging-tech/2020/03/white-house-releases-national-strategy-5g-security/164109/" target="_blank">a senior administration official</a>.</li>
<li><a href="https://twitter.com/kmkz_security/status/1244364245464162305" target="_blank">ZecOps teases CVE-2020-0769 for local privilege escalation</a>. The SMB v3 compression bug from the 2020-03-16 LWiS can also be leveraged for local privilege escalation. A Spanish researcher beat them to it with a <a href="https://github.com/danigargu/CVE-2020-0796" target="_blank">PoC</a> however. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1068/" target="_blank">T1068 Exploitation for Privilege Escalation</a>]</li>
<li><a href="https://portal.msrc.microsoft.com/en-us/security-guidance/advisory/adv200006" target="_blank">Type 1 Font Parsing Remote Code Execution Vulnerability</a> is a classic Windows remote code execution bug in the font parser, specifically <cite>atmfd.dll</cite>. This bug affects Adobe Type 1 PostScript format fonts, and is being actively exploited. Microsoft claims that Windows 10 is at low risk due to mitigations in the OS, but has manual mitigations for earlier operating systems.</li>
<li><a href="https://www.boozallen.com/c/insight/publication/the-logic-behind-russian-military-cyber-operations.html" target="_blank">The Logic Behind Russian Military Cyber Operations</a> is a well researched paper from Booz Allen Hamilton that details the public information related to 33 of the GRU's cyber operations at a high level. <a href="http://nullrefer.com/?https://www.boozallen.com/content/dam/boozallen_site/ccg/pdf/publications/bearing-witness-uncovering-the-logic-behind-russian-military-cyber-operations-2020.pdf" target="_blank">Direct link</a>.</li>
<li><a href="https://lists.zx2c4.com/pipermail/wireguard/2020-March/005206.html" target="_blank">WireGuard 1.0.0 for Linux 5.6 Released</a>. This fast UDP based VPN is much simpler than OpenVPN and now that it has reached 1.0 and is included by default in Linux I hope to see tools support it. While the VPN itself is great, there is no good solution for user and key management yet.</li>
</ul>
</section>
<section id="techniques">
<h2>Techniques</h2>
<ul>
<li><a href="https://dev.to/tauri/setting-up-ci-and-cd-for-tauri-48pp" target="_blank">Setting Up CI and CD for Tauri</a> is a writeup on how the Tauri team used GitHub actions to handle everything from vulnerability audit to release publishing and everything in between. <a href="https://tauri.studio/en" target="_blank">Tauri</a> itself is an interesting project looking to solve the systemic issues with cross-platform desktop apps like Electron.</li>
<li><a href="https://twitter.com/buffaloverflow/status/1242497902850854912" target="_blank">Arbitrary file DACL overwrite to SYSTEM shell</a>. This is a great trick that uses a named pipe with a path traversal string to turn a DACL overwrite into a primitive that can drop a DLL in system32 and then be called by <a href="https://github.com/itm4n/UsoDllLoader" target="_blank">UsoDllLoader</a>. C# version <a href="https://gist.github.com/rxwx/461fa06b3cf59bf1ae57658a4212262f" target="_blank">here</a>. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1068/" target="_blank">T1068 Exploitation for Privilege Escalation</a>]</li>
<li><a href="https://twitter.com/felixaime/status/1243083359212969984" target="_blank">MageCart is getting trickier</a> by faking a Google analytics script tag, and even serving the actual Google analytics javascript if the referrer header doesn't match a pwned site.</li>
<li><a href="https://twitter.com/osxreverser/status/1242944381751111680" target="_blank">Corellium ordered to hand over unredacted source code before 2020-03-31</a>. The iOS emulation company, in legal hot water for months, faces a new hurdle. I hope there can be a reasonable resolution to this, but based on the extreme measures Apple is pursuing, it doesn't look good.</li>
<li><a href="https://natedotred.wordpress.com/2020/03/28/cve-2020-8816-pi-hole-remote-code-execution/" target="_blank">CVE-2020-8816 – Pi-hole Remote Code Execution</a> is worth the read not for the exploit, but the technique. Nate walks through the challenges of character restricted command injection and creative use of POSIX shell parameter expansions to achieve remote code execution.</li>
<li><a href="https://en.hackndo.com/kerberos-asrep-roasting/" target="_blank">AS_REP Roasting</a> is a rarely seen but potentially powerful dangerous kerberos setting that allows hash dumping without any prior access or authentication. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1208/" target="_blank">T1208 Kerberoasting</a>]</li>
</ul>
</section>
<section id="tools-and-exploits">
<h2>Tools and Exploits</h2>
<ul>
<li><a href="https://github.com/FortyNorthSecurity/C2concealer" target="_blank">C2concealer</a>  is a command line tool that generates randomized C2 malleable profiles for use in Cobalt Strike. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1001/" target="_blank">T1001 Data Obfuscation</a>]</li>
<li><a href="https://github.com/outflanknl/TamperETW" target="_blank">TamperETW</a> is a 64 bit PoC based on the <a href="https://www.mdsec.co.uk/2020/03/hiding-your-net-etw/" target="_blank">blog post from last week</a> about hooking Windows ETW telemetry to hid .NET actions. This PoC blocks assembly load events by hooking EtwEventWrite using native system calls. EDR will likely have a tough time with this. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1054/" target="_blank">T1054 Indicator Blocking</a>]</li>
<li><a href="https://github.com/realoriginal/ppdump-public" target="_blank">ppdump-public</a> uses Zemana AntiMalware Engine To Open a Privileged Handle to a Privileged Process or Privileged Process Low (PPL) and inject MiniDumpWriteDump() shellcode. It even comes with an aggressor script for easy integration with Cobalt Strike. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1003/" target="_blank">T1003 Credential Dumping</a>]</li>
<li><a href="https://github.com/coffeegist/changeling" target="_blank">changeling</a> is a feature morphing tool that allows you to build dynamic payloads without having to constantly recompile. With correctly designed payload binaries, this tool can quickly swap out resources to change shellcode, settings, etc on the fly.</li>
<li><a href="https://github.com/0xZDH/redirect.rules" target="_blank">redirect.rules</a> dynamically generates a redirect.rules file that will redirect Sandbox environments away from a payload hosting/C2 server to a site of your choosing. It combines a ton of User-Agent rules and IP space for known malware analysis companies to help keep your payload undetected for longer. The output rules work on Apache 2.4+ but it would be fairly easy to convert the output to nginx or iptables block rules. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1090/" target="_blank">T1090 Connection Proxy</a>]</li>
<li><a href="https://github.com/m0bilesecurity/RMS-Runtime-Mobile-Security" target="_blank">Runtime Mobile Security</a> is a <a href="https://github.com/frida/frida" target="_blank">Frida</a> powered web interface for maniputlating Android Java Classes and Methods at runtime. A comparable iOS tool would be <a href="https://github.com/chaitin/passionfruit" target="_blank">passionfruit</a> and for a powerful CLI tool that supports both Android and iOS, check out <a href="https://github.com/sensepost/objection" target="_blank">objection</a>. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1055/" target="_blank">T1055 Process Injection</a>]</li>
<li><a href="https://www.tenable.com/security/research/tra-2020-15" target="_blank">Grandstream UCM62xx SQL Injection</a> - Tenable drops an unauthenticated remote code execution <a href="https://github.com/tenable/poc/blob/master/grandstream/ucm62xx/pbx_sploit.py" target="_blank">exploit</a> for the IP-PBX phone system. Over 10,000 of these show up on Shodan. Need more IoT exploits? Raelize released five for the end-of-life D-Link DSL-2640B <a href="https://raelize.com/posts/d-link-dsl-2640b-security-advisories/" target="_blank">here</a> including hard coded credentials and <a href="https://th0mas.nl/2020/03/26/getting-root-on-a-zyxel-vmg8825-t50-router/" target="_blank">Getting root on a Zyxel VMG8825-T50 router</a> is a great breakdown of the process from unboxing to root shell. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1190/" target="_blank">T1190 Exploit Public-Facing Application</a>]</li>
<li><a href="https://github.com/GhostPack/Lockless" target="_blank">Lockless</a> is a C# tool that allows for the enumeration of open file handles and the copying of locked files. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1005/" target="_blank">T1005 Data from Local System</a>]</li>
</ul>
</section>

<li><a href="https://github.com/wagoodman/dive" target="_blank">dive</a> is a tool for exploring each layer in a docker image interactively in a terminal user interface. It's great for showing what changes at each layer, and can be integrated with continuous integration to ensure space efficiency remains high.
