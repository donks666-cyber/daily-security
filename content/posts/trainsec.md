**TrainSec Academy** 是一家专注于高级网络安全培训的在线教育平台，由 Pavel Yosifovich、Uriel Kosayev、Amichai Yifrach 等资深讲师授课。课程涵盖 Windows 内部机制、内核编程、逆向工程、恶意软件分析、硬件黑客等多个方向。

---

## 学习路径

### Windows Master Developer
从普通 C 程序员成长为 Windows 用户态与内核态大师级开发者。
- 涵盖 Windows 系统编程、内核编程、COM、GUI 等
- [查看详情](https://trainsec.net/windows-master-developer/)

### Windows Internals Master
深入理解 Windows 内部运行机制，掌握进程、线程、内存管理、I/O 系统等核心概念。
- 5 门课程系列，从基础到深入
- [查看详情](https://trainsec.net/windows-internals-master/)

### Windows Security Researcher
掌握 Windows 操作系统安全研究所需的技能与工具。
- 包括 WinDbg、内核调试、漏洞分析等
- [查看详情](https://trainsec.net/windows-security-researcher/)

### Hardware Hacking Expert
从零基础成长为硬件黑客专家，涵盖 UART、I2C、SPI、JTAG、CAN Bus、蓝牙等通信协议的分析与利用。
- [查看详情](https://trainsec.net/hardware-hacking-expert-level-1/)
