<p>Last Week in Security is a summary of the interesting cybersecurity news, techniques, tools and exploits from the previous week. This post covers 2020-04-13 to 2020-04-20.
MITRE ATT&amp;CK techniques are in brackets where appropriate.</p>
<section id="news">
<h2>News</h2>
<ul>
<li><a href="https://github.blog/2020-04-14-github-is-now-free-for-teams/" target="_blank">GitHub is now free for teams</a> - private repositories with unlimited collaborators are now available to all GitHub accounts and the price of the paid plan drops to $4 per user per month. Microsoft is shoveling cash into the Github furnace to ensure dominance. To what end remains to be seen, but in the short term users benefit.</li>
<li><a href="https://hackerone.com/riot/" target="_blank">Riot Games offers $100,000 for kernel exploit in anti-cheat</a>. Riot Games (developers of league of legends) is offering a lot of money if anyone is able to execute code in the Windows kernel via their new "Vanguard" anti-cheat driver. Hacker One doesn't list a bounty if you can show "Vanguard" is being used as a backdoor; Riot Games is wholly owned by Chinese conglomerate holding company Tencent.</li>
<li><a href="https://twitter.com/proteaswang/status/1251722815843168256?s=21" target="_blank">tfp0 bug and exploit teased for iOS 13.4.1 on A13</a>. Big if true, this screenshot shows <cite>uname -a</cite> output for the latest iOS on the latest iPhone processor which means Qihoo 360 likely has a powerful iOS 0day on their hands.</li>
<li><a href="https://arxiv.org/pdf/2004.06195v1.pdf" target="_blank">AiR-ViBeR: Exfiltrating Data from Air-GappedComputers via Covert Surface ViBrAtIoNs</a> shows that there is yet another way to slowly leak information out of air-gapped networks, this time via vibrations caused by the variations of case fans and detected with an off the shelf cell phone on the same table. Demo video <a href="https://www.youtube.com/watch?v=XGD343nq1dg" target="_blank">here</a>. [<a class="m-text m-dim" href="https://attack.mitre.org/tactics/TA0009/" target="_blank">TA0009 Collection</a>]</li>
<li><a href="https://binary.ninja/2020/04/16/state-of-the-ninja-ep13.html" target="_blank">Binary Ninja adds a decompiler</a>. Hot on the heals of the Hex Ray's IDA Home announcement Binary Ninja adds a decompiler to their free cloud offering (graph view only) and offline disassemblers (graph view and linear). Keep up the great work Vector 35!</li>
<li><a href="https://arstechnica.com/gadgets/2020/04/caveat-emptor-smr-disks-are-being-submarined-into-unexpected-channels/" target="_blank">Buyer beware—that 2TB-6TB “NAS” drive you’ve been eyeing might be SMR</a> brings to light a rumor that has been gaining credibility. Despite being marketed as "NAS" drives, nearly all 2TB-6TB drives (yes - even WD Reds) are Shingled Magnetic Recording (SMR) drives. This has a huge impact on write speed and this technology was previously reserved for "archive" or "backup" drives. Seagate has confirmed that none of its IronWolf or IronWolf Pro drives use SMR, but are as cagey as the other major manufacturers about all other drives. This recent bout of shady practices is in the shadow of likely price fixing by the three major drive manufacturers since the Thailand flood of 2011. High capacity SSDs cannot come fast enough.</li>
<li><a href="https://flipperzero.one/zero" target="_blank">Flipper Zero hardware hacking tool announced</a>. This is a really cool looking piece of kit that, if they can deliver, will be an essential for every hacker's go-bag. It claims to have the capability to do everything from being a 433/868 MHz transceiver, 125kHz RFID cloner, InfraRed transceiver, Bad USB, iButton cloner, and have compatibility with the Arduino IDE. Big promises but I will be in line as soon as the kickstarter opens in May.</li>
</ul>
</section>
<section id="techniques">
<h2>Techniques</h2>
<ul>
<li><a href="https://posts.specterops.io/methodology-for-static-reverse-engineering-of-windows-kernel-drivers-3115b2efed83" target="_blank">Methodology for Static Reverse Engineering of Windows Kernel Drivers</a> takes the reader through the process of identifying drivers on targets, setting up a Ghidra environment to work with Windows drivers (setting up the symbols needed for analyzing drivers), finding the driver entry, and reversing functions. <a href="https://www.n4r1b.com/posts/2020/04/dissecting-the-windows-defender-driver-wdfilter-part-4/" target="_blank">n4r1b's blog</a> has even more Windows driver reversing.</li>
<li><a href="https://shenaniganslabs.io/2020/04/14/Internal-DNS-C2.html" target="_blank">DNS Peer-to-Peer Command and Control with ADIDNS</a> is a method of using Active Directory-Integrated DNS Zones (ADIDNS) records in restrictive corporate networks to bypass locked down outbound firewalls. Adding an ADIDNS entry (available to any authenticated domain user), tunneling with a <a href="https://gist.github.com/eladshamir/97161caa718b95160fa3b603edcfbc2a" target="_blank">helper C# tool</a>, and a little socat allows a Cobalt Strike beacon to relay though another Cobalt Strike beacon via DNS. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1048/" target="_blank">T1048 Exfiltration Over Alternative Protocol</a>]</li>
<li><a href="https://en.hackndo.com/constrained-unconstrained-delegation/" target="_blank">Kerberos Delegation</a> - Hackndo is back with another great article on Active Directory, this time focusing on the different types of kerberos delegation.</li>
<li><a href="https://www.mdsec.co.uk/2020/04/designing-the-adversary-simulation-lab/" target="_blank">Designing The Adversary Simulation Lab</a> by Adam Chester is a deep dive into the tools and technologies MDSec choose to build a deployable lab for their adversary emulation course and contains some insights on desired state configuration, terraform, and the intricacies of AWS. Adam even provides a <a href="https://github.com/xpn/DemoLab" target="_blank">demo lab</a>!</li>
<li><a href="https://medium.com/@polarply/build-your-first-llvm-obfuscator-80d16583392b" target="_blank">Build your first LLVM Obfuscator</a>. Ever wanted to venture into the depths of the LLVM compiler's intermediate representation to obfuscate a binary without changing the source code? This article introduces LLVM and walks through a string obfuscator. This technique could be expanded and used on open source red team tools as part of an AV/EDR bypass. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1027/" target="_blank">T1027 Obfuscated Files or Information</a>]</li>
</ul>
</section>
<section id="tools-and-exploits">
<h2>Tools and Exploits</h2>
<ul>
<li><a href="https://objective-see.com/products/blockblock_beta.html" target="_blank">BlockBlock 1.0 Beta</a> is an open source rewrite of the persistence monitor for macOS that uses the Endpoint Security Framework. If you are using the 0.9.x BlockBlock, you will have to manually uninstall and install this version. [<a class="m-text m-dim" href="https://attack.mitre.org/tactics/TA0003/" target="_blank">TA0003 Persistence</a>]</li>
<li><a href="https://github.com/kgretzky/pwndrop" target="_blank">pwndrop</a> is a self-deployable file hosting service for red teamers, allowing to easily upload and share payloads over HTTP and WebDAV from the maker of evilginx2. The UI is beautiful and the feature road map looks good. Read more on his <a href="https://breakdev.org/pwndrop/" target="_blank">blog</a>. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1192/" target="_blank">T1192 Spearphishing Link</a>]</li>
<li><a href="https://github.com/artssec/burp-exporter" target="_blank">burp-exporter</a> is a Burp Suite extension to copy a request to the clipboard as multiple programming languages functions.</li>
<li><a href="https://github.com/assafmo/xioc" target="_blank">xioc</a> extracts indicators of compromise from text, including "escaped" ones like hxxp://banana.com, 1.1.1[.]1 and phish at malicious dot com. This is a useful tool for automating "threat intelligence" pipelines.</li>
<li><a href="https://github.com/kris-anderson/remove-zoom-macos" target="_blank">remove-zoom-macos</a> - Zoom's recent security woes have you thinking twice about that app install? This script removes everything Zoom put on your mac, even the things the official uninstaller leaves behind.</li>
<li><a href="https://github.com/FSecureLABS/Jamf-Attack-Toolkit" target="_blank">Jamf-Attack-Toolkit</a> is a suite of tools to facilitate attacks against the Jamf macOS management platform. Check out the accompanying <a href="https://labs.f-secure.com/blog/jamfing-for-joy-attacking-macos-in-enterprise/" target="_blank">blog post</a> and <a href="https://objectivebythesea.com/v3/talks/OBTS_v3_cHall_lRoberts.pdf" target="_blank">slides</a>. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1133/" target="_blank">T1133 External Remote Services</a>]</li>
<li><a href="https://github.com/magisterquis/meshmembers" target="_blank">meshmembers</a> is a tool to organize a mesh network of redirectors and allow the state of the network to be actively maintained by each node. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1188/" target="_blank">T1188 Multi-hop Proxy</a>]</li>
<li><a href="https://github.com/guardicore/vmware_vcenter_cve_2020_3952" target="_blank">vmware_vcenter_cve_2020_3952</a> is an exploit for last week's CVE-2020-3952 in vCenter 6.7 that allows an unauthenticated attacker to add themselves as an Administrator to a vCenter if it was upgraded from 6.5 or earlier to 6.7 (fresh installs not affected). [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1190/" target="_blank">T1190 Exploit Public-Facing Application</a>]</li>
<li><a href="https://github.com/dirkjanm/ROADtools" target="_blank">ROADtools</a> is an Azure AD exploration framework (Rogue Office 365 and Azure (active) Directory tools). It currently contains a great recon tool with an Angular UI for exploring an Azure AD. Blog and stream <a href="https://dirkjanm.io/introducing-roadtools-and-roadrecon-azure-ad-exploration-framework/" target="_blank">here</a>.</li>
<li><a href="https://github.com/CCob/SweetPotato" target="_blank">SweetPotato</a> is a rewrite of JuciyPotato (Local Service to SYSTEM privilege escalation from Windows 7 to Windows 10 / Server 2019) that is now compatible with execute-assembly with some extras. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1068/" target="_blank">T1068 Exploitation for Privilege Escalation</a>]</li>
<li><a href="https://github.com/trustedsec/quicksql" target="_blank">quicksql</a> is a simple MSSQL query tool that allows you to connect to MSSQL databases and does not require administrative level rights to use.</li>
</ul>
</section>
<section id="new-to-me">
<h2>New to Me</h2>
<p>This section is for news, techniques, and tools that weren't released last week but are new to me. Perhaps you missed them too!</p>
<ul>
<li><a href="https://github.com/matterpreter/OffensiveCSharp" target="_blank">OffensiveCSharp</a> is a great collection of offensive C# tooling that can be used post-exploitation on Windows targets. Check the <a href="https://github.com/matterpreter/OffensiveCSharp/blob/master/README.md" target="_blank">readme</a> for a description of each tool.</li>
<li><a href="https://github.com/brimsec/brim" target="_blank">Brim</a> is a desktop application to efficiently search large packet captures and Zeek logs. It loads pcaps much faster than wireshark but allows detailed analysis of flows in wireshark with a single click.</li>
<li><a href="https://github.com/claudiodangelis/qrcp" target="_blank">qrpc</a> allows you to transfer files over wifi from your computer to your mobile device by scanning a QR code without leaving the terminal. It's bi-directional and can receive files from a phone with a handy web uploader.</li>
</ul>
