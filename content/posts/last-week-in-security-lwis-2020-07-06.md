<p>Last Week in Security is a summary of the interesting cybersecurity news, techniques, tools and exploits from the previous week. This post covers 2020-06-29 to 2020-07-06.
MITRE ATT&amp;CK techniques are in brackets where appropriate.</p>
<section id="news">
<h2>News</h2>
<ul>
<li><a href="https://www.lookout.com/documents/threat-reports/us/lookout-uyghur-malware-tr-us.pdf" target="_blank">Mobile APT Surveillance Campaigns Targeting Uyghurs</a>. iOS security company Lookout releases this detail-packed 52-page report detailing a years-long hacking campaign.</li>
<li><a href="https://www.vice.com/en_us/article/3aza95/how-police-took-over-encrochat-hacked" target="_blank">How Police Secretly Took Over a Global Phone Network for Organized Crime</a>. French authorities managed to push malware to the endpoints of Encrochat, an encrypted phone system used by criminals. This is also a case for or against encryption backdoors, depending on how you see such a backdoor being used. Either way, competitors are already in place to take over from Encrochat, including <a href="https://omertadigital.com/" target="_blank">Omerta</a>.</li>
<li><a href="https://nakedsecurity.sophos.com/2020/06/30/ios-14-flags-tiktok-53-other-apps-spying-on-iphone-clipboards/" target="_blank">iOS 14 flags TikTok, 53 other apps spying on iPhone clipboards</a>. The new iOS 14 beta has stronger clipboard controls which have exposed apps like TikTok and <a href="https://www.theverge.com/2020/7/4/21313214/reddit-code-clipboard-privacy-copy-ios" target="_blank">Reddit</a> for reading anything in a user's clipboard.</li>
<li><a href="https://cturt.github.io/freedvdboot.html" target="_blank">FreeDVDBoot - Hacking the PlayStation 2 through its DVD player</a>. While the technology (PlayStation 2) isn't very relevant today, the technique used to play homebrew discs without modifying the PlayStation at all is very impressive (IFO parsing buffer overflow and multi-stage payload).</li>
<li><a href="https://www.elastic.co/blog/elastic-security-opens-public-detection-rules-repo" target="_blank">Elastic Security opens public detection rules repo</a>. The company behind Elasticsearch (and owners of Endgame EDR) release their detection rules. These are in yet another repository, and only work with the closed source (and very expensive) Elastic SIEM. For a more open, cross-platform solution, stick to <a href="https://github.com/Neo23x0/sigma" target="_blank">sigma</a> rules.</li>
</ul>
</section>
<section id="techniques">
<h2>Techniques</h2>
<ul>
<li><a href="https://palant.info/2020/07/06/dismantling-bullguard-antivirus-online-protection/" target="_blank">Dismantling BullGuard Antivirus’ online protection</a>. <a href="https://twitter.com/WPalant" target="_blank">@WPalant</a> continues his tour of antivirus-browser interaction research with a seriously facepalm-able bypass and XSS for BullGuard. User education and sane browsing habits plus EDR are a much better combo than trying to put antivirus in the browser.</li>
<li><a href="https://www.directdefense.com/when-multi-factor-authentication-isnt-enough-bypassing-mfa-via-phishing/" target="_blank">When Multi-Factor Authentication Isn’t Enough – Bypassing MFA via Phishing</a> walks through the install and use of <a href="https://github.com/kgretzky/evilginx2" target="_blank">evilginx2</a> for MFA phishing against Okta.</li>
<li><a href="https://ezqelusia.blogspot.com/2020/06/some-dos-bugs-while-processing.html" target="_blank">Some DOS bugs while processing Microsoft LNK files</a> is a good post not for the bugs themselves but for the process of how the bugs were found and the discussion of pe-afl vs Winafl.</li>
<li><a href="https://lockboxx.blogspot.com/2020/07/using-binject.html" target="_blank">Using Binject</a> - Binject is not new, but this writeup by <a href="https://twitter.com/1njection" target="_blank">@1njection</a> shows the power of <span class="strike">a fully armed and operational Death Star</span> Binject and its potential for use in a <a href="https://github.com/gen0cide/gscript" target="_blank">gscript</a> payload. For adversary emulation engagements using this in a phishing payload to spread persistence would be particularly nasty. This project from the <a href="https://www.symbolcrash.com/" target="_blank">Symbol Crash</a> team is very impressive.</li>
<li><a href="https://research.nccgroup.com/2020/07/04/experiments-in-extending-thinkst-canary-part-1/" target="_blank">Experiments in Extending Thinkst Canary – Part 1</a>. "Deception" is a buzzword used to sell cyber solutions but it actually does work when implemented properly. EDR can give alerts that may or may not be tied to malicious activity, but if someone tries to exploit a honeypot that is 100% malicious. The signal-to-noise or confidence level of well implemented honeypot alerts is invaluable. In this post, <a href="https://twitter.com/ollieatnccgroup" target="_blank">@ollieatnccgroup</a> discusses how he extends the open source Python Thinkst <a href="https://github.com/thinkst/opencanary/" target="_blank">opencanary</a> project to implement custom honeypot services.</li>
<li><a href="https://research.securitum.com/art-of-bug-bounty-a-way-from-js-file-analysis-to-xss/" target="_blank">Art of bug bounty: a way from JS file analysis to XSS</a>. This is a great post as it shows the full background and process that went into finding this specific bug.</li>
<li><a href="https://labs.sentinelone.com/living-off-windows-land-a-new-native-file-downldr/" target="_blank">Living Off Windows Land – A New Native File "downldr"</a>. A new "LOLBin" for downloading arbitrary files on Windows 10. Lots of environments are monitoring for the well known certutil.exe LOLBin being used as a downloader, but I doubt many are looking for desktopimgdownldr.exe.</li>
<li><a href="https://posts.specterops.io/going-4-a-run-eb263838b944" target="_blank">GOing 4 A Run</a> introduces a shellcode runner written in Go that implements a few evasion techniques (shellcode encryption, block DLLs, parent process spoofing). Borrow these techniques (and maybe add <a href="https://github.com/C-Sto/BananaPhone" target="_blank">direct syscalls</a>) for your custom in-house PE loader!</li>
<li><a href="https://blog.two06.info/Reading-Windows-Sticky-Notes/" target="_blank">Reading Windows Sticky Notes</a>. Everyone has seen passwords written on sticky notes stuck to a monitor on an assessment, but what if target stores sensitive information on digital sticky notes as well? <a href="https://twitter.com/two06" target="_blank">@two06</a> releases a <a href="https://github.com/two06/SharpStick" target="_blank">SharpStick</a> to read sticky notes on Windows.</li>
<li><a href="https://www.securityinbits.com/malware-analysis/interesting-tactic-by-ratty-adwind-distribution-of-jar-appended-to-signed-msi/" target="_blank">Interesting tactic by Ratty &amp; Adwind for distribution of JAR appended to signed MSI</a>. Taking advantage of how MSI and JARs are read, Ratty &amp; Adwind actors manage to slip some JARs past signature validation using a simple but effective technique. Security-in-bits includes a Yara rule in this post to detect this technique.</li>
<li><a href="https://www.redcursor.com.au/blog/bypassing-crowdstrike-endpoint-detection-and-response" target="_blank">Bypassing CrowdStrike Endpoint Detection and Response</a> uses XOR payload encryption/brute-forcing and NtQueueApcThread to get around CrowdStrike's detection of Process Hollowing. Well done; it's rare to see these write-ups as red-teamers don't want to burn their methods once they get one to work.</li>
</ul>
</section>
<section id="tools-and-exploits">
<h2>Tools and Exploits</h2>
<ul>
<li><a href="https://github.com/rapid7/metasploit-framework/pull/13807" target="_blank">BIG-IP TMUI Directory Traversal and File Upload RCE (CVE-2020-5902)</a>. If you have any BIG-IP products you might already be compromised as public exploits dropped soon after the CVE announcement. More details <a href="https://research.nccgroup.com/2020/07/05/rift-f5-networks-k52145254-tmui-rce-vulnerability-cve-2020-5902-intelligence/" target="_blank">here</a>. Sigma rule <a href="https://github.com/Neo23x0/sigma/blob/master/rules/web/web_cve_2020_5902_f5_bigip.yml" target="_blank">here</a> but it only works if your BIG-IP devices are pushing logs to your SIEM as the exploit traffic is encrypted. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1190/" target="_blank">T1190 Exploit Public-Facing Application</a>]</li>
<li><a href="https://www.randori.com/cve-2020-2021-pan-os-advisory-poc/" target="_blank">CVE-2020-2021: Post Exploit Analysis</a>. Breathing easy after the BIG-IP/F5 news because you are running Palo Alto gear? Bad news, PAN-OS has a critical SAML signature verification bypass option available that if in place (unchecked) allows successful arbitrary logins. You know its bad when US CYBERCOM <a href="https://twitter.com/CNMF_CyberAlert/status/1277674547542659074" target="_blank">tweets</a> about it.</li>
<li><a href="https://github.com/ctxis/DLLHSC" target="_blank">DLLHSC</a> is a tool to generate leads and automate the discovery of candidates for DLL Search Order Hijacking by Context. Read their <a href="https://www.contextis.com/en/blog/dll-search-order-hijacking" target="_blank">article</a> about how it came to be and how it can be used. SpecterOps also has a new post on DLL Hijacking: <a href="https://posts.specterops.io/automating-dll-hijack-discovery-81c4295904b0" target="_blank">Automating DLL Hijack Discovery</a>.</li>
<li><a href="https://research.checkpoint.com/2020/apache-guacamole-rce/" target="_blank">Would you like some RCE with your Guacamole?</a>. Checkpoint releases details of an exploit that requires access to a machine that the Guacamole server connects to (i.e. compromised workstation or insider threat) which can be used to exploit the server and take over arbitrary remote desktop sessions. In this age of remote work, there are likely many large vulnerable instances.</li>
<li><a href="https://research.nccgroup.com/2020/07/02/technical-advisory-macos-installer-local-root-privilege-escalation-cve-2020-9817/" target="_blank">Technical Advisory – macOS Installer Local Root Privilege Escalation (CVE-2020-9817)</a>. This is a permission issue during the install of a package where a malicious user (or actor that has compromised a machine with only user-level access) could gain root access if a package is installed by an administrator while the exploit is running in the background. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1068/" target="_blank">T1068 Exploitation for Privilege Escalation</a>]</li>
<li><a href="https://lapcatsoftware.com/articles/disclosure2.html" target="_blank">Disclosure: Another macOS privacy protections bypass</a>. The lax checks on code signatures allows malicious actors to copy signed and entitled applications, gut them - modifying their resources, and use them to access otherwise "protected" resources (i.e. Desktop, Contacts, ~/Library/Safari/*, etc).</li>
<li><a href="https://secret.club/2020/07/01/diagtrack.html" target="_blank">Windows Telemetry service elevation of privilege</a> . While no compile-and-pwn proof of concept exists, the details in the post plus <a href="https://github.com/thesecretclub/diagtrack/blob/master/example.cpp" target="_blank">this bit of code</a> should get you most of the way there. [<a class="m-text m-dim" href="https://attack.mitre.org/techniques/T1068/" target="_blank">T1068 Exploitation for Privilege Escalation</a>]</li>
<li><a href="https://github.com/nccgroup/nccfsas/tree/main/Tools/bof-vs-template" target="_blank">bof-vs-template</a> is a template project for building Cobalt Strike BOFs in Visual Studio. There is also <a href="https://github.com/dtmsecurity/bof_helper" target="_blank">bof_helper</a> from dtmsecurity if you prefer Python.</li>
<li><a href="https://github.com/samtay/so" target="_blank">so</a> is a terminal interface for StackOverflow written in Rust. Save yourself a few clicks and get answers in your terminal.</li>
<li><a href="https://github.com/halfmoonui/halfmoon" target="_blank">halfmoon</a> is a front-end framework with a built-in dark mode, designed for rapidly building beautiful dashboards and product pages. I love the fact its designed to work without JavaScript (like this site)!</li>
<li><a href="https://github.com/FlameOfIgnis/Pwdb-Public" target="_blank">Pwdb-Public</a> is a repository of modern, frequently used passwords. If you are still using rockyou.txt, it's time to upgrade.</li>
</ul>
</section>
<section id="new-to-me">
<h2>New to Me</h2>
<p>This section is for news, techniques, and tools that weren't released last week but are new to me. Perhaps you missed them too!</p>
<ul>
<li><a href="https://github.com/maxgoedjen/secretive" target="_blank">Secretive</a> is an app for storing and managing SSH keys in the Secure Enclave. It is inspired by the sekey project, but rewritten in Swift with no external dependencies and with a handy native management app. If you have a Mac or manage Macs this is another level of protection for SSH keys, and Macs without a secure enclave can use other sources such as a Yubikey.</li>
<li><a href="https://github.com/Velocidex/velociraptor" target="_blank">velociraptor</a> is a tool for collecting host based state information using Velocidex Query Language (VQL) queries. A simple to deploy single binary agent and server for incident response.</li>
</ul>
